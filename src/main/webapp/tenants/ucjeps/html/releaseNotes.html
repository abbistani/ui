Replace a non-existent pluginRepository reference."
  • Update build scripts to support JDBC options in all db urls (CSPACE-6404)
    • Added support for JDBC options throughout the build and configuration files.
  • Refobjs call with page size 0 hangs (CSPACE-6406)
    • Fix infinite loop in LazyAuthorityRefDocList.java when page size is 0.
  • Clear Voucher Label Requests batch job only clears 40 vouchers at a time (UCBG-383)
    • Retrieve all records in batch job searches, instead of the default 40.
  • Other changes
    • Transfer changes from defaults into tenant overrides.
