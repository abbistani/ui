UC Berkeley CollectionSpace Release Notes

Version ucjeps_4.0-13 (July 29, 2014)

This release of CollectionSpace addresses the following issues:

  • Dimension unit field in Cataloging and Media templates should appear to the immediate right of the dimension value field (CSPACE-4638)
    • Reorder fields in dimensions table.
  • Add value to taxon name source dropdown (UCJEPS-533)
    • Add Rebman and O'Brien Checklist option to taxon termSource.
  • Other changes
    • Add deployment release version to footer.
    • Add deployment version to login page.
    • Add release notes page.
    • Implement changes from defaults in ucjeps tenant overrides.
    • Set up UCB version numbering and release notes.
    • Correct a comment in releaseNotes.html, so that it can be located by the make-release script.
    • In computed fields, allow specifying argument paths from the document root.
    • Fix spurious modified warnings caused by computed fields.
    • Add tags required for dimension nuxeo artifacts to be generated from the app layer.
    • Disable generation of dimension nuxeo artifacts from the services layer.
    • Add support for ui-type='richtext'.
    • Add .gitattributes, don't convert line endings in ckeditor files.
    • Add ckeditor library files.
    • Create rich text editor component.
    • Prevent clicking on ckeditor toolbar buttons from triggering page modified warning.
    • Add cleanup of event listeners, and convert nbsp entity to space.
    • Add rich text editor js files to record.html.
    • Add rich text editor styles.
    • Fix problems with richtext fields in repeatables, and fix styling issues.
    • Add support for ranges separated by em and en dashes.
    • Add other lexer rule, and allow question mark at end of oneDisplayDate.
    • Add support for year-month-day.
